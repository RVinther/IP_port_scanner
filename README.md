# IP- og port-scanner ![Python](https://img.shields.io/badge/Python-Script-yellow)

Et Python-script, der hurtigt tester om en enhed er online og om bestemte porte er åbne.

Scriptet giver dig en enkel menu i terminalen, hvor du kan pinge IP-adresser og lave simple scanninger af porte.

## Formål med projektet

At anvende Python til praktisk netværksfejlfinding og styrke min forståelse for, hvordan netværk fungerer i praksis.

Projektet kombinerer teori om IP, TCP, porte og forbindelser med et simpelt CLI-værktøj, der kan bruges i hverdagen som datatekniker.


## Funktioner

- Ping af valgfri IP-adresse (giver online/offline-status)  
- Scan én eller flere porte på en given IP  
- Validering af IP-adresser og portnumre  
- Tydelig status på hver port (ÅBEN/LUKKET)
